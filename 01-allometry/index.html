1  Allometry

Overview

Allometric equations represent the proportional and scaling relationships between different tree dimensions, such as the relationship between a treeâ€™s diameter and its height, biomass, or crown size. These relationships translate tree diameter measurements into biomass estimates, forming the foundation of forest carbon accounting. This chapter addresses uncertainty quantification in processes of allometric model selection, parameter estimation, correction of measurement error and predictive bias.

1.1 The Allometric Equation

\[ AGB = \alpha \times DBH^{\beta} \]

Or in logarithmic formâ€¦

\[ \ln(AGB) = \ln(\alpha) + \beta \times \ln(DBH) + \epsilon \]

Where:

  • AGB: Aboveground biomass (kg)
  • DBH: Diameter at breast height (cm)
  • Î±, Î²: Allometric parameters (species/biome-specific)
  • Îµ: Random error term

Allometric equations predict aboveground biomass from diameter measurements using species or biome-specific parameters. Uncertainty compounds from three main sources: model selection (tropical vs. temperate equations, species-specific vs. generic), parameter estimation (regression standard errors, sample sizes), and field measurements (DBH precision Â±0.5-1.0 cm, clinometer errors, wood density variability). Log-transformation adds complexity through back-transformation bias that must be corrected to avoid systematically underestimating biomass.

1.1.0.1 ARTâ€™s Allometric Exemption

ART-TREES V2.0 Section 8 excludes structural uncertainty in allometric models from uncertainty deductions when models are applied consistently between baseline and crediting periods. Since systematic bias cancels in net change calculations, you only need to quantify random error: parameter estimation uncertainty, measurement precision, and sampling uncertainty when scaling from plots to landscapes.

This creates a strategic advantage: reduce uncertainty deductions by maintaining consistent model application across time periods, upgrading from IPCC Tier 1 defaults to country-specific Tier 2 parameters, and investing in measurement precision rather than destructive sampling. Letâ€™s implement this with Monte Carlo simulation.

1.2.1 Equivalence Testing Framework

Objective: Determine if published equations are valid for local conditions

Method: Test if difference between predicted and observed biomass falls within acceptable threshold

Equivalence criteria (CEOS/NASA protocol): - Relative difference < Â±25% - Systematic bias < Â±5% - RMSE within 1.5Ã— calibration dataset RMSE

# Function: Equivalence test for allometric models
equivalence_test <- function(observed_biomass, predicted_biomass, threshold = 0.25) {
  
  # Calculate relative difference
  relative_diff <- (predicted_biomass - observed_biomass) / observed_biomass
  
  # Mean relative difference (systematic bias)
  mean_rel_diff <- mean(relative_diff, na.rm = TRUE)
  
  # Standard error
  se_rel_diff <- sd(relative_diff, na.rm = TRUE) / sqrt(length(relative_diff))
  
  # 90% confidence interval
  ci_lower <- mean_rel_diff - 1.645 * se_rel_diff
  ci_upper <- mean_rel_diff + 1.645 * se_rel_diff
  
  # Equivalence test: CI must fall entirely within [-threshold, +threshold]
  is_equivalent <- (ci_lower > -threshold) && (ci_upper < threshold)
  
  # RMSE
  rmse <- sqrt(mean((predicted_biomass - observed_biomass)^2, na.rm = TRUE))
  
  # Results
  results <- list(
    mean_relative_difference = mean_rel_diff,
    ci_90 = c(ci_lower, ci_upper),
    is_equivalent = is_equivalent,
    rmse = rmse,
    n_samples = length(observed_biomass)
  )
  
  return(results)
}

1.3.2 Monte Carlo Parameter Resampling

Approach: Sample from multivariate normal distribution of parameter estimates

library(MASS)  # For mvrnorm

# Function: Monte Carlo parameter uncertainty propagation
monte_carlo_allometry <- function(model_fit, new_dbh, n_sim = 10000) {
  
  # Extract parameter estimates
  param_est <- coef(model_fit)
  
  # Extract variance-covariance matrix
  param_vcov <- vcov(model_fit)
  
  # Sample parameters from multivariate normal
  param_samples <- mvrnorm(n = n_sim, mu = param_est, Sigma = param_vcov)
  
  # Initialize results
  biomass_predictions <- matrix(NA, nrow = n_sim, ncol = length(new_dbh))
  
  # Baskerville correction factor
  sigma_model <- summary(model_fit)$sigma
  baskerville <- exp(sigma_model^2 / 2)
  
  # Monte Carlo simulation
  for (i in 1:n_sim) {
    alpha_sim <- param_samples[i, 1]
    beta_sim <- param_samples[i, 2]
    
    # Predict biomass with sampled parameters
    ln_biomass_pred <- alpha_sim + beta_sim * log(new_dbh)
    
    # Back-transform with Baskerville correction
    biomass_predictions[i, ] <- exp(ln_biomass_pred) * baskerville
  }
  
  # Calculate statistics
  biomass_mean <- colMeans(biomass_predictions)
  biomass_sd <- apply(biomass_predictions, 2, sd)
  biomass_ci_lower <- apply(biomass_predictions, 2, quantile, 0.05)
  biomass_ci_upper <- apply(biomass_predictions, 2, quantile, 0.95)
  
  # Uncertainty as percentage
  biomass_uncertainty_pct <- (biomass_ci_upper - biomass_ci_lower) / (2 * biomass_mean) * 100
  
  results <- data.frame(
    dbh_cm = new_dbh,
    biomass_mean_kg = biomass_mean,
    biomass_sd_kg = biomass_sd,
    ci_lower_kg = biomass_ci_lower,
    ci_upper_kg = biomass_ci_upper,
    uncertainty_pct = biomass_uncertainty_pct
  )
  
  return(list(
    summary = results,
    simulations = biomass_predictions
  ))
}

1.4 Error Propagation

1.4.1 DBH Measurement Precision

Sources of DBH error: 1. Instrument precision: Â±0.5 cm (diameter tape) to Â±1.0 cm (calipers) 2. Measurement height: Breast height (1.3m) Â±5 cm 3. Tree form irregularities: Buttresses, branches at 1.3m 4. Operator variability: Inter-measurer differences

# Function: Propagate DBH measurement error through allometry
propagate_dbh_error <- function(true_dbh, dbh_error_sd, alpha, beta, n_sim = 10000) {
  
  # Sample DBH measurements with error
  dbh_measurements <- rnorm(n_sim, mean = true_dbh, sd = dbh_error_sd)
  
  # Ensure positive values
  dbh_measurements <- pmax(dbh_measurements, 0.1)
  
  # Calculate biomass for each measurement
  biomass_estimates <- alpha * dbh_measurements^beta
  
  # Summary statistics
  biomass_mean <- mean(biomass_estimates)
  biomass_sd <- sd(biomass_estimates)
  biomass_cv <- biomass_sd / biomass_mean * 100
  biomass_ci <- quantile(biomass_estimates, c(0.05, 0.95))
  
  # True biomass (no error)
  biomass_true <- alpha * true_dbh^beta
  
  # Bias from measurement error
  bias_pct <- (biomass_mean - biomass_true) / biomass_true * 100
  
  results <- list(
    true_dbh = true_dbh,
    mean_biomass = biomass_mean,
    true_biomass = biomass_true,
    bias_percent = bias_pct,
    cv_percent = biomass_cv,
    ci_90 = biomass_ci
  )
  
  return(results)
}

1.5 Bias Correction

1.5.1 The Baskerville Correction

Problem: Log-transformation bias in back-transformation

Uncorrected (WRONG): \[ \hat{B} = \exp(\hat{\alpha} + \hat{\beta} \times \ln(DBH)) \]

Systematically underestimates biomass!

Baskerville-corrected (CORRECT): \[ \hat{B} = \exp\left(\hat{\alpha} + \hat{\beta} \times \ln(DBH) + \frac{\sigma^2}{2}\right) \]

Where \(\sigma^2\) = residual variance from log-scale regression

library(tidyverse)
library(MASS)

# Function: Complete allometry uncertainty assessment
complete_allometry_uncertainty <- function(
  calibration_data,
  validation_data,
  dbh_error_sd = 0.5,
  n_sim = 10000
) {
  
  # Step 1: Fit allometric model
  calibration_data$ln_dbh <- log(calibration_data$dbh_cm)
  calibration_data$ln_agb <- log(calibration_data$agb_kg)
  
  model_fit <- lm(ln_agb ~ ln_dbh, data = calibration_data)
  
  # Step 2: Extract parameters and uncertainty
  param_est <- coef(model_fit)
  param_vcov <- vcov(model_fit)
  sigma_residual <- summary(model_fit)$sigma
  baskerville <- exp(sigma_residual^2 / 2)
  
  # Step 3: Monte Carlo simulation
  biomass_sims <- matrix(NA, nrow = n_sim, ncol = nrow(validation_data))
  
  for (i in 1:n_sim) {
    # Sample parameters
    params_sim <- mvrnorm(1, mu = param_est, Sigma = param_vcov)
    alpha_sim <- params_sim[1]
    beta_sim <- params_sim[2]
    
    # Sample DBH with measurement error
    dbh_sim <- rnorm(nrow(validation_data), 
                     mean = validation_data$dbh_cm, 
                     sd = dbh_error_sd)
    dbh_sim <- pmax(dbh_sim, 0.1)  # Ensure positive
    
    # Predict biomass with Baskerville correction
    ln_biomass_pred <- alpha_sim + beta_sim * log(dbh_sim)
    biomass_sims[i, ] <- exp(ln_biomass_pred) * baskerville
  }
  
  # Step 4: Calculate statistics
  results_summary <- data.frame(
    dbh_cm = validation_data$dbh_cm,
    biomass_mean_kg = colMeans(biomass_sims),
    biomass_sd_kg = apply(biomass_sims, 2, sd),
    ci_lower_kg = apply(biomass_sims, 2, quantile, 0.05),
    ci_upper_kg = apply(biomass_sims, 2, quantile, 0.95)
  ) %>%
    mutate(
      uncertainty_pct = (ci_upper_kg - ci_lower_kg) / (2 * biomass_mean_kg) * 100,
      hw_90 = (ci_upper_kg - ci_lower_kg) / 2
    )
  
  # Step 5: Validation metrics
  if ("observed_agb_kg" %in% colnames(validation_data)) {
    results_summary$observed_kg <- validation_data$observed_agb_kg
    results_summary$bias_pct <- (results_summary$biomass_mean_kg - results_summary$observed_kg) / results_summary$observed_kg * 100
    
    overall_rmse <- sqrt(mean((results_summary$biomass_mean_kg - results_summary$observed_kg)^2))
    overall_bias <- mean(results_summary$bias_pct)
  } else {
    overall_rmse <- NA
    overall_bias <- NA
  }
  
  return(list(
    summary = results_summary,
    simulations = biomass_sims,
    model_fit = model_fit,
    rmse = overall_rmse,
    mean_bias_pct = overall_bias

## The Allometric Equation {#sec-allometry-relationship}

$$
AGB = \alpha \times DBH^{\beta}
$$

^[*Or in logarithmic form...*]{.smallcaps}^

$$
\ln(AGB) = \ln(\alpha) + \beta \times \ln(DBH) + \epsilon
$$

Where:

-   AGB: Aboveground biomass (kg)
-   DBH: Diameter at breast height (cm)
-   Î±, Î²: Allometric parameters (species/biome-specific)
-   Îµ: Random error term

Allometric equations predict aboveground biomass from
diameter measurements using species or biome-specific
parameters. Uncertainty compounds from three main sources:
model selection (tropical vs. temperate equations,
species-specific vs. generic), parameter estimation
(regression standard errors, sample sizes), and field
measurements (DBH precision Â±0.5-1.0 cm, clinometer errors,
wood density variability). Log-transformation adds
complexity through back-transformation bias that must be
corrected to avoid systematically underestimating biomass.

#### ART's Allometric Exemption

ART-TREES V2.0 Section 8 excludes structural uncertainty in
allometric models from uncertainty deductions when models
are applied consistently between baseline and crediting
periods. Since systematic bias cancels in net change
calculations, you only need to quantify random error:
parameter estimation uncertainty, measurement precision, and
sampling uncertainty when scaling from plots to landscapes.

This creates a strategic advantage: reduce uncertainty
deductions by maintaining consistent model application
across time periods, upgrading from IPCC Tier 1 defaults to
country-specific Tier 2 parameters, and investing in
measurement precision rather than destructive sampling.
Let's implement this with Monte Carlo simulation.

### Equivalence Testing Framework {#sec-equivalence-testing}

Objective: Determine if published equations are valid for
local conditions

Method: Test if difference between predicted and observed
biomass falls within acceptable threshold

Equivalence criteria (CEOS/NASA protocol): - Relative
difference \< Â±25% - Systematic bias \< Â±5% - RMSE within
1.5Ã— calibration dataset RMSE

```{r}
library(tidyverse)
library(MASS)

# Function: Equivalence test for allometric models
equivalence_test <- function(observed_biomass, predicted_biomass, threshold = 0.25) {
  
  # Calculate relative difference
  relative_diff <- (predicted_biomass - observed_biomass) / observed_biomass
  
  # Mean relative difference (systematic bias)
  mean_rel_diff <- mean(relative_diff, na.rm = TRUE)
  
  # Standard error
  se_rel_diff <- sd(relative_diff, na.rm = TRUE) / sqrt(length(relative_diff))
  
  # 90% confidence interval
  ci_lower <- mean_rel_diff - 1.645 * se_rel_diff
  ci_upper <- mean_rel_diff + 1.645 * se_rel_diff
  
  # Equivalence test: CI must fall entirely within [-threshold, +threshold]
  is_equivalent <- (ci_lower > -threshold) && (ci_upper < threshold)
  
  # RMSE
  rmse <- sqrt(mean((predicted_biomass - observed_biomass)^2, na.rm = TRUE))
  
  # Results
  results <- list(
    mean_relative_difference = mean_rel_diff,
    ci_90 = c(ci_lower, ci_upper),
    is_equivalent = is_equivalent,
    rmse = rmse,
    n_samples = length(observed_biomass)
  )
  
  return(results)
}
```

```{r}
library(MASS)  # For mvrnorm

# Function: Monte Carlo parameter uncertainty propagation
monte_carlo_allometry <- function(model_fit, new_dbh, n_sim = 10000) {
  
  # Extract parameter estimates
  param_est <- coef(model_fit)
  
  # Extract variance-covariance matrix
  param_vcov <- vcov(model_fit)
  
  # Sample parameters from multivariate normal
  param_samples <- mvrnorm(n = n_sim, mu = param_est, Sigma = param_vcov)
  
  # Initialize results
  biomass_predictions <- matrix(NA, nrow = n_sim, ncol = length(new_dbh))
  
  # Baskerville correction factor
  sigma_model <- summary(model_fit)$sigma
  baskerville <- exp(sigma_model^2 / 2)
  
  # Monte Carlo simulation
  for (i in 1:n_sim) {
    alpha_sim <- param_samples[i, 1]
    beta_sim <- param_samples[i, 2]
    
    # Predict biomass with sampled parameters
    ln_biomass_pred <- alpha_sim + beta_sim * log(new_dbh)
    
    # Back-transform with Baskerville correction
    biomass_predictions[i, ] <- exp(ln_biomass_pred) * baskerville
  }
  
  # Calculate statistics
  biomass_mean <- colMeans(biomass_predictions)
  biomass_sd <- apply(biomass_predictions, 2, sd)
  biomass_ci_lower <- apply(biomass_predictions, 2, quantile, 0.05)
  biomass_ci_upper <- apply(biomass_predictions, 2, quantile, 0.95)
  
  # Uncertainty as percentage
  biomass_uncertainty_pct <- (biomass_ci_upper - biomass_ci_lower) / (2 * biomass_mean) * 100
  
  results <- data.frame(
    dbh_cm = new_dbh,
    biomass_mean_kg = biomass_mean,
    biomass_sd_kg = biomass_sd,
    ci_lower_kg = biomass_ci_lower,
    ci_upper_kg = biomass_ci_upper,
    uncertainty_pct = biomass_uncertainty_pct
  )
  
  return(list(
    summary = results,
    simulations = biomass_predictions
  ))
}
```

```{r}
propagate_dbh_error <- function(true_dbh, dbh_error_sd, alpha, beta, n_sim = 10000) {
  
  # Sample DBH measurements with error
  dbh_measurements <- rnorm(n_sim, mean = true_dbh, sd = dbh_error_sd)

  dbh_measurements <- pmax(dbh_measurements, 0.1)
  
  # Calculate biomass for each measurement
  biomass_estimates <- alpha * dbh_measurements^beta
  
  # Summary statistics
  biomass_mean <- mean(biomass_estimates)
  biomass_sd <- sd(biomass_estimates)
  biomass_cv <- biomass_sd / biomass_mean * 100
  biomass_ci <- quantile(biomass_estimates, c(0.05, 0.95))
  
  # True biomass (no error)

    true_dbh = true_dbh,
    mean_biomass = biomass_mean,
    true_biomass = biomass_true,
    bias_percent = bias_pct,
    cv_percent = biomass_cv,
    ci_90 = biomass_ci
  )
  
  return(results)
}
```

## Bias Correction {#sec-bias-correction}

### The Baskerville Correction {#sec-baskerville}

Problem: Log-transformation bias in back-transformation

Uncorrected (WRONG): $$
\hat{B} = \exp(\hat{\alpha} + \hat{\beta} \times \ln(DBH))
$$

Systematically underestimates biomass!

Baskerville-corrected (CORRECT): $$
\hat{B} = \exp\left(\hat{\alpha} + \hat{\beta} \times \ln(DBH) + \frac{\sigma^2}{2}\right)
$$

Where $\sigma^2$ = residual variance from log-scale
regression

```{r}
#| eval: false
#| label: allometry-baskerville
#| code-summary: "Demonstrate Baskerville correction necessity"

library(tidyverse)

# Simulated calibration data
set.seed(123)
n_trees <- 50
calibration_sim <- data.frame(
  dbh_cm = runif(n_trees, 5, 50)
)

# True biomass with lognormal error
calibration_sim$true_biomass_kg <- 0.05 * calibration_sim$dbh_cm^2.5 * exp(rnorm(n_trees, 0, 0.2))

# Fit log-transformed model
calibration_sim$ln_dbh <- log(calibration_sim$dbh_cm)
calibration_sim$ln_biomass <- log(calibration_sim$true_biomass_kg)

model_log <- lm(ln_biomass ~ ln_dbh, data = calibration_sim)

# Extract parameters
alpha_hat <- coef(model_log)[1]
beta_hat <- coef(model_log)[2]
sigma_residual <- summary(model_log)$sigma

# Predict on new data
prediction_data <- data.frame(dbh_cm = seq(5, 50, by = 1))

# UNCORRECTED prediction
prediction_data$biomass_uncorrected <- exp(alpha_hat + beta_hat * log(prediction_data$dbh_cm))

# BASKERVILLE-CORRECTED prediction
baskerville_factor <- exp(sigma_residual^2 / 2)
prediction_data$biomass_corrected <- exp(alpha_hat + beta_hat * log(prediction_data$dbh_cm)) * baskerville_factor

# TRUE biomass (without error)
prediction_data$biomass_true <- 0.05 * prediction_data$dbh_cm^2.5

# Calculate bias
prediction_data$bias_uncorrected_pct <- (prediction_data$biomass_uncorrected - prediction_data$biomass_true) / prediction_data$biomass_true * 100
prediction_data$bias_corrected_pct <- (prediction_data$biomass_corrected - prediction_data$biomass_true) / prediction_data$biomass_true * 100

# Visualize
ggplot(prediction_data, aes(x = dbh_cm)) +
  geom_line(aes(y = biomass_true, color = "True"), size = 1.2) +
  geom_line(aes(y = biomass_uncorrected, color = "Uncorrected"), linetype = "dashed", size = 1) +
  geom_line(aes(y = biomass_corrected, color = "Baskerville-Corrected"), size = 1) +
  scale_color_manual(
    values = c("True" = "black", "Uncorrected" = "red", "Baskerville-Corrected" = "blue"),
    name = "Prediction Method"
  ) +
  labs(
    title = "Importance of Baskerville Correction",
    subtitle = "Uncorrected back-transformation systematically underestimates biomass",
    x = "DBH (cm)",
    y = "Biomass (kg)"
  ) +
  theme_minimal()

# Summary statistics
cat("Mean bias (uncorrected):", round(mean(prediction_data$bias_uncorrected_pct), 2), "%\n")
cat("Mean bias (corrected):", round(mean(prediction_data$bias_corrected_pct), 2), "%\n")
```

Typical magnitude: Uncorrected models underestimate biomass
by 5-15% depending on residual variance.

### Monte Carlo Simulation {#sec-case-study-uncertainty}

Complete implementation for selected models:

```{r}
#| eval: false
#| label: allometry-complete-uncertainty
#| code-summary: "End-to-end uncertainty quantification for ART-TREES"

# =============================================================================
# ART-TREES COMPLIANT ALLOMETRY UNCERTAINTY ASSESSMENT
# Combines: Parameter uncertainty + Measurement error + Bias correction
# =============================================================================

library(tidyverse)
library(MASS)

# Function: Complete allometry uncertainty assessment
complete_allometry_uncertainty <- function(
  calibration_data,
  validation_data,
  dbh_error_sd = 0.5,
  n_sim = 10000
) {
  
  # Step 1: Fit allometric model
  calibration_data$ln_dbh <- log(calibration_data$dbh_cm)
  calibration_data$ln_agb <- log(calibration_data$agb_kg)
  
  model_fit <- lm(ln_agb ~ ln_dbh, data = calibration_data)
  
  # Step 2: Extract parameters and uncertainty
  param_est <- coef(model_fit)
  param_vcov <- vcov(model_fit)
  sigma_residual <- summary(model_fit)$sigma
  baskerville <- exp(sigma_residual^2 / 2)
  
  # Step 3: Monte Carlo simulation
  biomass_sims <- matrix(NA, nrow = n_sim, ncol = nrow(validation_data))
  
  for (i in 1:n_sim) {
    # Sample parameters
    params_sim <- mvrnorm(1, mu = param_est, Sigma = param_vcov)
    alpha_sim <- params_sim[1]
    beta_sim <- params_sim[2]
    
    # Sample DBH with measurement error
    dbh_sim <- rnorm(nrow(validation_data), 
                     mean = validation_data$dbh_cm, 
                     sd = dbh_error_sd)
    dbh_sim <- pmax(dbh_sim, 0.1)  # Ensure positive
    
    # Predict biomass with Baskerville correction
    ln_biomass_pred <- alpha_sim + beta_sim * log(dbh_sim)
    biomass_sims[i, ] <- exp(ln_biomass_pred) * baskerville
  }
  
  # Step 4: Calculate statistics
  results_summary <- data.frame(
    dbh_cm = validation_data$dbh_cm,
    biomass_mean_kg = colMeans(biomass_sims),
    biomass_sd_kg = apply(biomass_sims, 2, sd),
    ci_lower_kg = apply(biomass_sims, 2, quantile, 0.05),
    ci_upper_kg = apply(biomass_sims, 2, quantile, 0.95)
  ) %>%
    mutate(
      uncertainty_pct = (ci_upper_kg - ci_lower_kg) / (2 * biomass_mean_kg) * 100,
      hw_90 = (ci_upper_kg - ci_lower_kg) / 2
    )
  
  # Step 5: Validation metrics
  if ("observed_agb_kg" %in% colnames(validation_data)) {
    results_summary$observed_kg <- validation_data$observed_agb_kg
    results_summary$bias_pct <- (results_summary$biomass_mean_kg - results_summary$observed_kg) / results_summary$observed_kg * 100
    
    overall_rmse <- sqrt(mean((results_summary$biomass_mean_kg - results_summary$observed_kg)^2))
    overall_bias <- mean(results_summary$bias_pct)
  } else {
    overall_rmse <- NA
    overall_bias <- NA
  }
  
  return(list(
    summary = results_summary,
    simulations = biomass_sims,
    model_fit = model_fit,
    rmse = overall_rmse,
    mean_bias_pct = overall_bias
  ))
}

# Example usage with Model M3 (Betula pendula)
# Calibration data (from published equation)
calibration_m3 <- data.frame(
  dbh_cm = seq(5, 40, by = 5),
  agb_kg = exp(-2.7584 + 2.6134 * log(seq(5, 40, by = 5)))
)

# Validation data (field measurements)
validation_m3 <- data.frame(
  dbh_cm = c(10, 15, 20, 25, 30, 35),
  observed_agb_kg = c(12, 35, 68, 110, 165, 235)
)

# Run complete assessment

)
```

## ART Uncertainty Deduction {#sec-allometry-art-trees}

Equation 11 [@artArchitectureREDDTransactions2021, pp. 46]

$$
UA_t = 0.524417 \times \frac{HW_{90\%}}{1.645006}
$$

Example calculation:

```{r}
#| eval: false
#| label: allometry-uncertainty-deduction

  plot_id = 1:3,
  biomass_mean_Mg_ha = c(86.94, 112.59, 260.21),
```

## Notes on Next Steps {#sec-allometry-summary}

For \<10% allometry uncertainty: 1. Use geographically
proximate equations (same biome/country) 2. Validate with
â‰¥50 independent trees per species 3. Apply Baskerville
correction to log-transformed models 4. Quantify DBH
measurement error (Â±0.5 cm target) 5. Conduct Monte Carlo
simulation (n=10,000)

For 10-20% uncertainty (acceptable Tier 2): - Generic
pantropical/temperate equations - Validation with 25-50
trees - Standard regression uncertainty quantification

For \>20% uncertainty (Tier 1, high deductions): - IPCC
default biomass expansion factors (BEFs) - No validation
data - Generic equations from distant regions
