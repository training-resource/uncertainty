Modelling sources vs. recurring sources of uncertainty

“The highest error propagated using Monte Carlo simulations was caused by modelling sources of uncertainty, which calls for special attention to ensure consistency in REDD+ reporting which is essential for securing environmental integrity.” (Pelletier et al., 2024)

Strategic implication: Modelling sources can be standardized across jurisdictions (one-time investment), while recurring sources require continuous monitoring (ongoing costs).

Sensitivity Analysis Framework

Sobol indices identify variance contributions:

First-order effects (Si): Direct contribution

Total effects (STi): Direct + interactions

Typical results: Activity data (Si = 0.42) >> Biomass (0.31) >> Combustion factor (0.18) >> Emission factor (0.03)

Complementary methods:

Partial correlation coefficients

Tornado diagrams for parameter ranking

Panama case study (Pelletier et al., 2024): Fallow classification and carbon accumulation = highest sensitivity

Cost-benefit ranking for uncertainty reduction (Chapter 4 detail)

Temporal aggregation provision:

5-year recalculation: Aggregate uncertainty across crediting period

Recovery mechanism: Over-deducted credits issued if cumulative uncertainty decreases

Typical benefit: 5-10 percentage point reduction from temporal averaging

Example: Individual-year deductions = 492,500 t CO2; Aggregated = 406,700 t CO2 → Recovery: 85,800 t CO2

Key Terminology

Precision vs. Accuracy:

Precision: Reproducibility of measurements (random error)

Accuracy: Closeness to true value (systematic bias)

Confidence intervals:

90% CI: Range containing true value with 90% probability

Half-width: Distance from mean to CI bound = (Upper - Lower) / 2

Error types:

Type 1: False rejection of null hypothesis (α error)

Type 2: False acceptance of null hypothesis (β error)

Type 3: Solving the wrong problem (asking the wrong question)

Type 4: Applying results outside valid domain (extrapolation)

IPCC Tiers:

Tier 1: IPCC default methods and parameters

Tier 2: Country-specific methods or parameters

Tier 3: Higher-order methods (models, repeated measurements)

LULC Approaches:

Approach 1: Total area change (no spatial data)

Approach 2: Tracking of land-use conversions (transition matrices)

Approach 3: Spatially explicit tracking (wall-to-wall maps)
